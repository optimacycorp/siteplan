# Honest Assessment

The 3-month goal is realistic only if this is a planning/demo MVP, not a survey-grade production system.

Realistic in 3 months:
- 3DEP/USGS contour overlay
- address-first search
- parcel-ID secondary search where a provider supports it
- building-footprint auto-load from open data/fixtures
- persistent labels for bearings, distances, area, and notes
- a misclosure/Entry Tool wedge for typed calls
- basic deed-call entry that generates a sketch plat
- limited rule-table-based setback/constraint overlay

Not realistic in 3 months:
- universal legal-description parsing
- legally reliable plat generation
- nationwide parcel normalization
- complete zoning automation
- survey-grade closure determinations

Best framing:
A lightweight site-plan and deed-sketch assistant for planning exhibits, early due diligence, and drafting support — not a replacement for survey, title, or zoning review.
